What is the DOM?
 The DOM stands for Docuument Object Model and it's a structured
representation of HTML documents. It allows Javascript to access
HTML elements and styles to manipulate them.

How is the DOM structured?
 The DOM has a tree like structure which has all of the HTML elements that are within the given document
