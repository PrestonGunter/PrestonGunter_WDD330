Function Review

There are 3 main function types that can be used in javascript.
1. Function Declaration: Functions that can be used before it's declared.
2. Function Expression: Essentially a function value stored in a variable.
3. Arrow Function: Great for a quick one-line functions. Has no this keyword.

Array Review

There are two main ways to make an array.
Examples:
const friends = ['Preston', 'Kendra', 'Mario'];
const years = new Array(1456, 1998, 1995, 2020);
Most of the time when making arrays you use the first one but both of them work the same way.
You can put what ever you want into an array ints, strings, floats.
