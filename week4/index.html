    Dot vs Bracket Notation

    There are two ways to call properties from and object.
    Example object.property object[property]
    With the bracket notation you can do a whole bunch of stuff thats pretty cool.
    for an example take a look a the console to see my examples.
    When we need to first compute the property name we use the bracket notation.
    In any other case we use the dot notation.
